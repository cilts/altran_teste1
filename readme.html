Como Funciona?

Você pode através dos link do Menu, verificar os dados imputados nos PostTypes e suas respectivas CustomFields.

Para administrar, acesse o painel administrativo (wp-admin) utilizando o usuário: altran e senha: altran.

No menu principal, existe os sub menus Produtos, Clientes e Pedidos.

Esses sub menus, podem ser desativados atraés do plugin Altran.

Os Custom Fields foram criados através do plugin ACF (Advanced Custom Fields).
